# Yuvilearn Website

Static website for Yuvilearn, an AI-powered learning platform focused on NEET foundation students.

## Current pages

- `index.html` — landing page
- `login.html` — login/sign-up entry
- `dashboard.html` — student dashboard mock

## SEO files

- `robots.txt`
- `sitemap.xml`

## Deploy notes

The project includes `vercel.json` for static hosting on Vercel.
