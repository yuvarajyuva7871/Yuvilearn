# Yuvilearn Website

Static website for Yuvilearn, an AI-powered learning platform focused on NEET foundation students.

## Current pages

- `index.html` — home page with product overview and navigation
- `signup.html` — student account creation flow
- `login.html` — student login flow (email/password + Google)
- `dashboard.html` — authenticated student dashboard with AI doubt chat

## Backend/API

- `server.js` — Express API with `/ask-ai` endpoint for Gemini-powered doubt support
- `firebase.js` — Firebase client initialization for authentication

## SEO files

- `robots.txt`
- `sitemap.xml`

## Deploy notes

The project includes `vercel.json` for static hosting on Vercel.
